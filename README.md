# Birthday Surprise Website 🎉

A festive, interactive website to surprise someone on their birthday with a special experience!

## Features

- **Interactive Landing Page**: Fun YES/NO buttons with a playful dodge effect
- **Photo Gallery**: Beautiful slideshow of birthday memories
- **Birthday Letter**: Warm message for the special person
- **Reasons Page**: Reasons to celebrate them
- **Final Surprise**: A final birthday wish and celebration
- **Celebration Page**: Confetti, balloons, and sparkles

## Structure

```
birthday-surprise/
├── index.html              # Landing page
├── pages/                  # All interior pages
├── css/                    # Stylesheets
├── js/                     # JavaScript files
├── assets/                 # Images, audio, icons
└── README.md
```

## Setup Instructions

1. **Add Your Photos**:
   - Replace placeholder files in `assets/images/` with your actual photos
   - Recommended: img1.jpg, img2.jpg, img3.jpeg, cover.png

2. **Add Music** (Optional):
   - Replace the sample track in `assets/audio/love-song.mp3` with your birthday song

3. **Customize Content**:
   - Edit the birthday letter in `pages/letter.html`
   - Update reasons in `pages/reasons.html`
   - Personalize messages throughout the site

4. **Deploy**:
   - Upload to web hosting or GitHub Pages
   - Share the link with the birthday person!

## How to Use

1. Open `index.html` in a web browser
2. Click YES to begin the journey (NO button will run away!)
3. Navigate through the pages to experience the surprise

## Technologies Used

- HTML5
- CSS3 (with animations)
- Vanilla JavaScript
- SVG icons

## Customization Tips

- **Colors**: Edit CSS variables in `css/style.css`
- **Animations**: Modify `css/animations.css`
- **Content**: Update text in HTML files
- **Gallery**: Edit image paths and captions in `js/gallery.js`
